Pairing rule cheat-sheet:
  • CTAs & links: Amber Rust only.
  • Metric numerals & "verified"/"live" labels: Eucalyptus only.
  • Section eyebrows: one voice per section — Rust for action-led, Eucalyptus for insight-led.
  • Outlines: Cloud Zinc by default; Rust = active/focused; Eucalyptus = validated.
  • Tags: Rust-light = interactive category; Eucalyptus-light = metadata / status.
  • Icons: Rust = action (arrow, plus, link); Eucalyptus = insight (check, shield, trend).
  • Never mix both accents inside a single component. Voltage comes from separation.
